# 💰 AI Finance Tracker

An intelligent **AI-powered Finance Tracker** built with **Python**, **Streamlit**, and the **Groq API** that allows users to upload **CSV** and **Excel** files for financial analysis. The application automatically processes your financial data, generates insights, visualizes spending patterns, and answers finance-related questions using AI.

---

## 🚀 Features

* 📂 Upload **CSV** and **Excel (.xlsx)** financial files
* 🤖 AI-powered financial analysis using the **Groq API**
* 💬 Interactive AI chatbot for finance-related queries
* 📊 Automatic charts and data visualizations
* 💵 Income and expense tracking
* 📈 Spending trend analysis
* 📋 Financial summaries and reports
* ⚡ Fast and user-friendly Streamlit interface
* 🔍 Intelligent insights from uploaded datasets

## 📦 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/AI-Finance-Tracker.git
```

### 2. Navigate to the Project Folder

```bash
cd AI-Finance-Tracker
```

### 3. Create a Virtual Environment (Optional)

```bash
python -m venv venv
```

Activate it:

**Windows**

```bash
venv\Scripts\activate
```

**Mac/Linux**

```bash
source venv/bin/activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Add Your Groq API Key

Create a `.env` file:

```env
GROQ_API_KEY=your_groq_api_key_here
```

### 6. Run the Application

```bash
streamlit run app.py
```

## 📊 How It Works

1. Launch the Streamlit application.
2. Upload your financial data in **CSV** or **Excel** format.
3. The application processes the dataset automatically.
4. View financial summaries and interactive charts.
5. Ask the integrated AI assistant questions about your finances.
6. Receive intelligent insights and recommendations generated by the Groq LLM.
